Clematis Storage API is one of the low-level microservices, useful
for the rest of applications which would like to store files. The client for Spring
is now deployed to GitHub packages:

```xml
<dependency>
    <groupId>org.clematis.storage</groupId>
    <artifactId>storage-api-client</artifactId>
    <version>1.0.2</version>
</dependency>
```

Documentation is [ready](/docs/web-applications/capabilities-sharing)
