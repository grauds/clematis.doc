[Clematis Desktop](https://github.com/grauds/clematis.desktop)
is updated to work with Java 25.

The project gradually accepts [new language features](https://docs.oracle.com/en/java/javase/25/language/java-language-changes-summary.html), so it won't be compiling with 
the earlier JDKs soon.
